Project SbFake

SbFAKE is an innovative machine learning algorithm, born in the Northwestern Security & AI lab to counter Intellectual Property (IP) theft. This innovate solution uses a unique combination of language processing , optimization, and the psychological notion of surprise to create counterfeit variants of technical documents. This strategic approach aims to deter cyber attackers intent on IP theft, a threat estimated to cost the US economy between $225 billion to $600 billion annually, constituting 1% to 5% of the GDP.

My role in this project was to develop a user-friendly web interface for SbFAKE. This interface enables users to generate counterfeit documents with customizable parameters, compare fakes and originals, and contribute to algorithm enhancement by suggesting substitution replacements.

Collaboration and User-Centric Design

The first step in this project involved collaborative and iterative design sessions with the research team behind SbFAKE reviewing designs and brainstorming ideas. Our shared vision focused on enabling users to efficiently examine both the authentic document and its counterfeit counterpart within a single page, enhancing the user experience.Â 

Enhanced User Engagement

To maintain user involvement during the algorithm's runtime, a challenge given its duration of 30-60 minutes, I harnessed the capabilities of two libraries, Celery and Celery-Progress. These tools expertly managed asynchronous tasks and provided real-time progress updates to the React Client.

This effectively enhanced the usability of our tool and addressed potential obstacles our users could have encountered.

Robust and Secure Backend

The SbFAKE project demanded a robust and secure backend to address cybersecurity concerns comprehensively. To meet these requirements, I implemented a REST API using the Django DRF framework, fortified with session authentication and served with Gunicorn. Further bolstering security, NGINX served as a reverse proxy to host both the client and server. The entire system was deployed on a private RHEL8 remote server, ensuring a secure and efficient environment

We presented a live demo of the SbFake interface and algorithm at the National Security and AI conference hosted by the Buffett Instute for Global Affairs and NSAIL however this tool is not yet publicly available as we are continuously improving the algorithm and interface with the insights we gathered.
